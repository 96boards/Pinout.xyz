GPIO-L

|Signal        |Description |V      |Type|
| ------------- |:--------------:| -----:|------:|
|GPIO-L     |General Purpose I/O | 1.8V |I/O |

##Special Function:

GPIO-L has an alternate function for CSI control if required by the display/sensor.

|Alternate Description |V  	   |	Type|
| -------------------- |:-----:| ------:|
|CSI1_PWDN    		   |1.8V   | Output |
