GPIO-F

|Signal        |Description |V      |Type|
| ------------- |:--------------:| -----:|------:|
|GPIO-F     |General Purpose I/O | 1.8V |I/O |

##Special Function:

GPIO-F has an alternate function for DSI control if required by the display/sensor.

|Alternate Description |V  	   |	Type|
| -------------------- |:-----:| ------:|
|DSI_BLCTL     		   |1.8V   | Output |
