GPIO-K

|Signal        |Description |V      |Type|
| ------------- |:--------------:| -----:|------:|
|GPIO-K     |General Purpose I/O | 1.8V |I/O |


##Special Function:

GPIO-K has an alternate function for CSI control if required by the display/sensor.

|Alternate Description |V  	   |	Type|
| -------------------- |:-----:| ------:|
|CSI1_RST     		   |1.8V   | Output |
