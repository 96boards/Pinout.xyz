GPIO-J

|Signal        |Description |V      |Type|
| ------------- |:--------------:| -----:|------:|
|GPIO-J     |General Purpose I/O | 1.8V |I/O |

##Special Function:

GPIO-J has an alternate function for CSI control if required by the display/sensor.

|Alternate Description |V  	   |	Type|
| -------------------- |:-----:| ------:|
|CSI0_PWDN    		   |1.8V   | Output |
