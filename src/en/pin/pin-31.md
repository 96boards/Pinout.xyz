GPIO-I

|Signal        |Description |V      |Type|
| ------------- |:--------------:| -----:|------:|
|GPIO-I     |General Purpose I/O | 1.8V |I/O |

##Special Function:

GPIO-I has an alternate function for CSI control if required by the display/sensor.

|Alternate Description |V  	   |	Type|
| -------------------- |:-----:| ------:|
|CSI0_RST    		   |1.8V   | Output |
